# 🌍 Wanderlust

A full-stack Airbnb-inspired web application that allows users to explore, create, and manage vacation rental listings with secure authentication, interactive maps, image uploads, wishlists, reviews, real-time chat, and AI-powered travel assistance.

## ✨ Features

- 🔐 Secure Authentication & Authorization
- 🏡 Create, Edit & Delete Listings
- 📷 Cloudinary Image Uploads
- 🗺️ Interactive Maps with Mapbox
- ❤️ Wishlist Management
- ⭐ Reviews & Ratings
- 💬 Real-time Chat using Socket.IO
- 🤖 AI Travel Assistant (Google Gemini)
- 📱 Fully Responsive UI

## 📂 Installation

```bash
git clone https://github.com/mahekdembla/wanderlust.git
cd wanderlust
npm install
```

Create a `.env` file:

```env
ATLASDB_URL=
SECRET=

CLOUD_NAME=
CLOUD_API_KEY=
CLOUD_API_SECRET=

MAP_TOKEN=
GEMINI_API_KEY=
```

Run the project:

```bash
npm start
```

Visit:

```
http://localhost:8080
```
